# Plaza React Components

React component library for Plaza Design System.

## Installation

Peer dependency: `react ^18.0.0 || ^19.0.0`

```tsx
import 'plaza-design-core/plaza.css';
import { Window, Button } from 'plaza-react-components';
```

## Components

| Component | Description |
|-----------|-------------|
| `Window` | Terminal window frame with title bar |
| `Panel` | Side panel container |
| `PanelBlock` | Section within a panel |
| `Button` | Styled button with variants |
| `Modal` | Dialog overlay |
| `GridCanvas` | Animated perspective grid background |
| `Particles` | Floating particle effect |
