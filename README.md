# Plaza Design Kit

A cyberpunk terminal design system.

## What's Here

| File | Description |
|------|-------------|
| `plaza.css` | Standalone CSS design system |
| `plaza.js` | JS utilities (theme, clock, effects) |
| `demo-single.html` | All-in-one demo (no server needed) |
| `demo-modular.html` | Modular demo (requires local server) |
| `demo-react/` | React demo app |
| `design-guide/` | Design philosophy & guidelines |

## Quick Start

```html
<link rel="stylesheet" href="plaza.css">
<button class="plaza-btn">ENTER</button>
```

## Documentation

See [design-guide/](./design-guide/) for philosophy, hierarchy, language, and gimmicks.
